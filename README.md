🚀 Introducing JavaKart – My Java-Powered Console E‑Commerce App!

🔑 Key Features:
✅ User registration & login with strict validation — if the username or password is wrong, the app halts immediately.
🛍️ Product catalog sorted by ID using Comparable<Product>.
➕➖ Shopping cart: add by ID, remove items, view cart, and dynamic total updates.
✅ Console-based order placement with confirmation flow.
☕ Built using Java OOP and Collections, and input validation.

🎯 Challenges & learnings:
🔢 Modeled the domain with Comparable<Product> for easy sorting.
🎬 Implemented session‑like flow in a console environment.
💪 Strengthened skills in object‑oriented design, Collections, and exception handling.
